① 基本の流れ

麻酔モニタリングの場面で、このアプリは次の順で使うのを推奨します。

┌─────────────────────────┐
│ 1. 患者管理タブ          │ ← 患者を新規登録
│    （カルテ番号・名前・   │   （麻酔開始前にあらかじめ登録しておく）
│      体重・種別）        │
└────────────┬────────────┘
             ↓
┌─────────────────────────┐
│ 2. ヘッダーから患者選択   │ ← 「患者を選択」をタップして対象を指定
└────────────┬────────────┘
             ↓
┌─────────────────────────┐
│ 3. 撮影タブで撮影        │ ← 麻酔中にバイタルモニターを撮影
│    （カメラで一発）      │   AI が値を自動抽出 → DB に自動保存
└────────────┬────────────┘
             ↓
┌─────────────────────────┐
│ 4. 撮影記録タブで確認    │ ← その日の記録を時刻順に閲覧
│    （必要なら ✏ で修正） │   AI の読み取り誤りはここで補正
└────────────┬────────────┘
             ↓
┌─────────────────────────┐
│ 5. トレンドタブで時系列   │ ← 心拍数・血圧などをグラフで可視化
│    変化を確認            │
└─────────────────────────┘

⚠️ 3 の撮影時は 解析中（スピナー表示中）にリロードしない でください。サーバー処理が中断されると、その回の記録が保存されません。

② 各タブの使い方

🐾 患者管理タブ

患者の登録・編集・削除を行います。

┌──────────────────────────────┐
│ [＋ 新規患者登録]              │ ← 新規登録ボタン
├──────────────────────────────┤
│ ┌──────────────────────────┐ │
│ │ [犬] 田中 太郎 [選択中] ✏│ │ ← 患者カード
│ │ #12345 · 5.2 kg          │ │   ✏ で編集・削除
│ │              [選択] [✏] │ │   [選択]でヘッダーへ反映
│ └──────────────────────────┘ │
└──────────────────────────────┘
  • 新規登録: 名前（必須）・カルテ番号・体重・種別（犬/猫/その他）を入力して保存
  • 編集 / 削除: 患者カードの ✏ ボタンで編集モーダルを開く。削除もモーダル内から実行
  • 選択中バッジ: ヘッダーで現在選択している患者には [選択中] が付く
  • 対象患者の切り替え: 各カードの [選択] ボタンで切り替え（ヘッダーにも反映）

📷 撮影タブ

カメラでバイタルモニターを撮影し、AI で値を抽出します。撮影と同時に DB に自動保存されます。

┌──────────────────────────────┐
│ モニター種別 [機種指定なし▼]  │ ← 未指定でも OK（AI が自動判定）
│ [◉ 撮影開始]                  │ ← ボタンでカメラ起動
├──────────────────────────────┤
│ 基本バイタル                  │
│  心拍数   ---                │ ← 未撮影は「---」
│  P1（観血測定平均） ---       │
│  ...                         │
│ 呼吸器・麻酔器                │
│  ...                         │
│ 人工呼吸器                    │
│  ...                         │
└──────────────────────────────┘

撮影モーダル:
┌──────────────────────────────┐
│ 撮影            [✕]          │
│   カメラプレビュー            │
│   （解析中はスピナー）        │
│         ◯                   │ ← シャッターボタン
└──────────────────────────────┘
  1. モニター種別を選ぶ（不明な場合は「機種指定なし」のままで OK。AI が自動判定）
  2. 「撮影開始」でカメラが起動
  3. シャッターを押すと解析開始（数秒〜十数秒）
  4. 解析完了で値が画面に反映 + DB に自動保存される
  5. 「自動検出: 機種名」バッジが出れば、AI が機種識別したことを示す

⚠️ 解析中はリロード／タブクローズを避ける。離れようとするとブラウザの警告が出ます。DB 保存に失敗した場合はエラーバナーで通知されます。

📋 撮影記録タブ

その日の撮影記録を時刻順に一覧表示します。AI の読み取り誤りはここで修正できます。

┌──────────────────────────────┐
│ ‹  2026/05/21  ›             │ ← 日付ピッカー（前日・翌日）
├──────────────────────────────┤
│ ┌──────────────────────────┐ │
│ │ 14:32 [Bio-Scope AM140] ✏│ │ ← 記録カード（✏で編集）
│ │ [カルテ番号 12345] [体重 5.2 kg] │
│ │ 心拍数      75 bpm       │ │
│ │ ...                      │ │
│ └──────────────────────────┘ │
└──────────────────────────────┘
  • 日付の切り替え: 上部の日付ピッカー or ‹ › ボタンで前日・翌日へ
  • 患者でフィルタ: ヘッダーで患者選択中ならその患者の記録のみ表示
  • 記録の修正: 各カードの ✏ ボタンで全項目を編集
  • 記録の削除: 修正モーダル内の「削除」ボタン（要確認）
  • 後から患者紐付け: 修正モーダル先頭の「患者」セレクトで既存患者を選ぶと、名前・カルテ番号・体重が自動入力される

📈 トレンドタブ

複数項目を時系列で重ねてグラフ表示します。1 日単位で表示し、時間範囲を指定できます。

┌──────────────────────────────────┐
│ [✓心拍数][✓血圧(平均)][✓呼吸数]   │ ← 項目チップ（複数選択可）
│ [✓体温][P1][P2][SpO2][EtCO2]     │   変更すると自動で再描画
├──────────────────────────────────┤
│ ‹  2026/05/21  ›                 │ ← 日付ピッカー（前日・翌日）
│ 時間  [00:00] 〜 [23:59]         │ ← 同日内の時間範囲
│ [更新]                  [数値表示]│
├──────────────────────────────────┤
│ ╔════════════════════════╗       │
│ ║    折れ線グラフ         ║      │ ← Chart.js
│ ╚════════════════════════╝       │
├──────────────────────────────────┤
│              12:15  14:32  16:00 │ ← 列=時刻
│ 心拍数(bpm)    75    78    80    │ ← 行=項目
│ 体温(°C)      38.1  38.2  38.0   │
└──────────────────────────────────┘
  1. 項目チップをタップして表示項目を選択（複数可・変更で自動更新）
  2. 日付を選択（前日・翌日ボタンで切替、変更で自動更新）
  3. 時間範囲を指定（デフォルト 00:00 〜 23:59）→「更新」
  4. 「数値表示」ボタンでグラフ各点の値を ON/OFF
  5. テーブル: 1 列目（項目名）は横スクロール時も固定。1 行目（時刻）はスティッキー

③ その他のシナリオ

❓ 患者登録前に撮影してしまった場合

撮影は患者選択なしでも実行できます。後から記録を編集して患者に紐付けることができます。

[撮影タブ] 患者未選択のまま撮影
       ↓
[撮影記録タブ] その記録の ✏ をタップ
       ↓
[修正モーダル] 先頭の「患者」セレクトから既存患者を選択
       ↓
名前・カルテ番号・体重が自動入力 → 保存
       ↓
レコードが患者に紐付けられる

❓ AI の読み取り間違いを修正したい

撮影記録タブの ✏ ボタン → 修正モーダルで該当項目を編集して保存します。 数値を 空欄にすると null（---） として保存されます。

❓ 患者選択中に他の患者の記録も見たい

ヘッダーの患者選択行をタップ → モーダル上部の 「選択解除（全患者表示）」 をタップすると、全患者の記録が表示されます。

❓ トレンドで複数日のデータを見たい

現バージョンは 1 日単位での表示のみです。日付ピッカーの ‹ › ボタンで日を切り替えて確認してください。

❓ 撮影してもデータが保存されなかった

以下のいずれかが原因の可能性があります:

  • 解析中にリロード／タブクローズした → サーバー処理が中断され保存されていない
  • DB 接続エラー → 画面にエラーバナーが表示される。サーバー管理者に確認

解析中はブラウザの警告が出るので、画面に値が表示されるまでリロードしないでください。
